# To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Instant UI updates without page refresh
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
1. Clone the repository.
2. Open the project folder in VS Code.
3. Start the project using the Live Server extension.
4. Begin adding and managing your tasks.

## Future Enhancements
- Store tasks using Local Storage
- Edit existing tasks
- Add task priorities
- Dark mode support
